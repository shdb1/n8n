# n8n Pinecone RAG Chatbot Demo

## Overview
RAG chatbot using n8n, Pinecone, OpenAI Embeddings and AI Agent.

## Setup
1. Import workflow JSON into n8n.
2. Configure OpenAI credentials.
3. Configure Pinecone credentials.
4. Configure Google Drive credentials.
5. Create Pinecone index (1536 dimensions).
6. Run ingestion workflow.
7. Chat with the agent.

## Example Questions
- How many casual leaves are employees entitled to?
- What is the WFH policy?
- How many paternity leaves are allowed?
